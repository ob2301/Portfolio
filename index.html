Time Series Foundation Model Toolkit
Sep. 2025 - Jan. 2026

PyTorch, NumPy, StandardScalar

I contributed several core components to the Foundation Model Toolkit (FMTK), including full integration of the FordA time-series dataset for binary classification, where I achieved ~94% accuracy using the MOMENT backbone and later ~90–93% using my custom Mantis backbone wrapper. I also extended FMTK with complete support for the Epilepsy 4-class time-series dataset, initially reaching ~71% using Mantis + SVM, and ultimately boosting performance to ~99% with an optimized MLP decoder. All models can be trained and evaluated through FMTK’s high-level API in 7 lines of code or fewer, enabling rapid experimentation and benchmark-quality results with minimal boilerplate.

Crisis Engine Simulator
Oct. 2025 - Current

Agentic Workflows, PyTorch, TensorFlow, React, JavaScript, Docker

This is a project that I am currently working on, but I will describe the idea behind it. I built a live, AI-driven city simulation that models floods, emergency responders, traffic, and vulnerable zones in real time. It uses agentic workflows to let police, EMS, and civilians “think” and move on their own, and the UI shows everything on an interactive map with live metrics and graphs. The goal is to see how quickly a city could reach safety if a disaster hits, and which areas are in trouble even before it happens. It’s built with a mix of React, TypeScript, and some ML tooling under the hood.

AI Document Deobfuscator
Jun. 2025 - Aug. 2025

PyTorch, NumPy, LangChain, Guardrails, React, JavaScript, AWS Lambda, Docker

The AI Document Deobfuscator is a tool I built to clean up messy or partially unreadable documents using a mix of ML, LLM reasoning, and validation rules. It takes things like noisy PDFs or obfuscated text and reconstructs the important information in a clean, structured format. I used PyTorch, NumPy, LangChain, and Guardrails to make sure outputs stay accurate and compliant, and automated the whole thing with AWS Lambda + Docker. The end result is a pipeline that can turn really rough documents into something usable with almost no manual effort.
